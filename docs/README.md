# Docs

Siin kaustas asub aktiivne Markdown-dokumentatsioon, mida kasutame arenduse allikana.

## Kaustad

- `architecture/` sihtarhitektuur, tehniline tegevuskaart, andmeskeem, mooduliregister ja repo struktuur
- `api/` API spetsifikatsioon, payload-valideerimine ja ekspordikihi kirjeldus
- `product/` õppeloogika, analüütika ning õpetaja vaate kontseptsioonid
- `decisions/` olulised tehnilised otsused
- `plans/` ehitusetapid, bootstrap, workflow, autosave, testimine, monitooring ja go-live valmisolek

## Märkus

Osa siia paigutatud dokumente on konverteeritud Wordi failidest ning võivad vajada sisulist toimetamist, täiendamist ja ühtlustamist enne kui neist saab lõplik “single source of truth”.

Wordi algdokumendid, hinnangud ja konversioonide arhiivid paiknevad `reference/` kaustas.
