# Hindamislehed

Siia kausta on koondatud Etapp 1A inimtaidetavad hindamislehed.

Failid:

- `stage_1A_master_checklist.md` - koondvaade kogu Etapp 1A jaoks
- `stage_1A_1_user_flow_assessment.md` - kasutajavoo hindamine
- `stage_1A_2_content_and_fields_assessment.md` - sisu ja valjade hindamine

1. Taida `stage_1A_1_user_flow_assessment.md`
2. Taida `stage_1A_2_content_and_fields_assessment.md`
3. Taida `stage_1A_3_validation_persistence_and_summary_assessment.md`
