# Reference

Siin kaustas on ajaloolised ja toetavad materjalid, mida hoiame alles, kuid ei kasuta aktiivse arenduse peamise tõeallikana.

## Kaustad

- `source-docs/` algsed Wordi spetsifikatsioonid ja etapidokumendid
- `project-evaluation/` hinnangud, traceability, QA materjalid ja nende Markdown-konversioonid
- `code-packs/` eraldi zip-pakid varasematest realiseeringutest
- `converted-md/archive/` ChatGPT abil loodud Markdown-konversioonide algsed zip-pakid
- `supporting-materials/` muud abimaterjalid

## Märkus

Kui mõni dokument muutub aktiivseks arendusallikaks, tuleks see tõsta `docs/` alla sobivasse alamkausta ja vajadusel ümber nimetada.
